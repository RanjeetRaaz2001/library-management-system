# Library Management System

A comprehensive web-based Library Management System with Admin and User roles.

## Features

### Admin Features
- Full system access
- Maintenance module
  - Add/Update Books & Movies
  - Add/Update Memberships
  - User Management
- Reports generation
- Transaction management

### User Features
- Book search and availability check
- Issue and return books
- Fine payment
- View reports

## Default Credentials

### Admin Login
- **User ID:** `adm`
- **Password:** `adm`

### User Login
- **User ID:** `user`
- **Password:** `user`

## System Requirements

- Modern web browser (Chrome, Firefox, Edge, Safari)
- JavaScript enabled
- Internet connection (for CDN resources)

## Installation

1. Clone or download the project
2. Extract to web server directory
3. Open `index.html` in browser
4. Login with default credentials
