Maintenance Guidelines

Adding Members

  • All fields are mandatory
  • Select membership duration: 6 months (default), 1 year, or 2 years
  • Aadhar Card number must be unique
  • Start date defaults to today

Managing Books

  • Select Book or Movie type
  • All fields are mandatory
  • Serial numbers auto-generated
  • Quantity defaults to 1

User Management

  • Select New User or Existing User
  • Name field is mandatory
  • Checkbox for Active status
  • Checkbox for Admin privileges
